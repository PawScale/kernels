# Metal Compute Kernels Examples

This directory contains example code demonstrating how to use the Metal Compute Kernels framework for GPU computing on Apple Silicon.

## Examples

### Vector Addition
[VectorAddition.swift](VectorAddition.swift) - Basic example showing how to add two vectors using GPU compute.

```swift
let a = [1.0, 2.0, 3.0, ...]
let b = [2.0, 3.0, 4.0, ...]
// Result: [3.0, 5.0, 7.0, ...]
```

### Matrix Multiplication
[VectorAddition.swift](VectorAddition.swift) - Example demonstrating matrix multiplication using Metal kernels.

## Running Examples

```bash
swift run
```
